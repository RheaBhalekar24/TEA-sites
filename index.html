Thesis Project

Designing a softer, more wearable knee exoskeleton.

This project investigates how twisted elastic actuators can replace rigid lower-limb actuation with a compact, compliant, and human-friendly alternative for assistive robotics.

Project Overview

Why this research matters

The challenge with conventional exoskeletons

Traditional lower-limb exoskeletons often depend on rigid motors, valves, and heavy mechanical structures. While effective in delivering support, these systems can feel bulky, restrictive, and difficult to use comfortably in everyday environments.

The goal of this thesis

The Soft Knee Exoskeleton project explores a more wearable solution by combining soft robotic principles with twisted elastic actuators. The aim is to make assistive devices feel more natural, compact, and practical for human movement.

Core Technology

What are twisted elastic actuators?

Twisted elastic actuators build on the idea of twisted string actuation, but replace rigid string materials with soft and elastic elements. This modification introduces compliance directly into the actuator and opens up new behavior for wearable systems.

Variable stiffness

Twisting can actively change the stiffness of the elastic cord, allowing the actuator to adapt to different tasks and interaction conditions.

Improved force control

The compliant elastic element behaves like a series elastic component, improving stability and precision during human-robot interaction.

Impact tolerance

Elasticity helps absorb shocks and reduce the load transmitted to motors, gearboxes, and other rigid drivetrain elements.

Energy storage and release

The actuator can store potential energy and release it during dynamic motion, supporting more efficient cyclic movement.

Reduced wear

By damping vibrations and impact loads, the elastic system can reduce drivetrain stress and extend component life.

Experimental Work

Testing the actuator behavior

The project includes tests examining force output with respect to the number of actuator turns and the number of strings used in the setup. These experiments help characterize performance, repeatability, and nonlinear behavior at higher twist levels.

Force vs. revolutions

The force response shows an approximately linear trend until the system reaches peak force. This gives a useful operating region for predictable assistance.

Repeatability

Repeatability tests indicate that the revolution count for consistent system behavior remains stable across multiple iterations.

Double coiling effects

After peak force, the force can decrease and then rise suddenly. This behavior is attributed to double coiling and is a key nonlinear phenomenon in the actuator response.
